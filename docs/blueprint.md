# **App Name**: G&G Auto Hub

## Core Features:

- Dynamic Inventory Display: Publicly browse and search two-wheelers by brand, model, price range, and type, presented in a clean, card-based layout.
- Detailed Vehicle Profiles: Dedicated pages for each vehicle, showcasing high-resolution image galleries, comprehensive specifications, condition details, and direct communication buttons.
- Secure Admin Panel: An authenticated dashboard for administrators to securely add, edit, delete vehicles, mark them as sold, and manage inquiries via Firebase Authentication.
- AI Vehicle Description Assistant: An administrative tool to generate compelling and detailed descriptions for vehicles based on input specifications, enhancing content quality and saving time.
- Lead Generation System: Integrated forms for 'Sell Your Bike', 'Exchange Inquiry', and general 'Contact Us', with all submission data securely stored in Firestore.
- Direct Communication Integrations: Seamless 'Click-to-call' and WhatsApp buttons directly on vehicle listings, featuring pre-filled messages for immediate and convenient customer contact.
- Performance & SEO Foundation: Optimized loading with lazy-loaded and responsive images, secure HTTPS hosting, and comprehensive meta-data implementation for robust search engine discoverability.

## Style Guidelines:

- Primary Color: Deep Blue (#0A3D91). This rich blue provides a foundation of trust and reliability, resonating with the business's mission of providing dependable vehicles.
- Background Color: A subtle, desaturated light blue (#E7EEF7). This bright, airy tone supports the clean and modern aesthetic, allowing content to stand out.
- Accent Color: Vibrant Orange (#FF7A00). As requested for buttons, this energetic hue provides excellent contrast with the primary blue, drawing attention to calls-to-action.
- Text Color: Charcoal (#333333). This dark grey provides clear readability against the light background, offering a soft contrast compared to pure black, for a modern, inviting feel.
- Headings Font: 'Poppins' (sans-serif) for a contemporary, precise feel, aligning with a modern design. Note: currently only Google Fonts are supported.
- Body Text Font: 'Montserrat' (sans-serif) for its clean lines and excellent readability, complementing 'Poppins' and reinforcing the modern aesthetic. Note: currently only Google Fonts are supported.
- Use a set of modern, flat-design icons that are clearly identifiable and support navigation and feature presentation without adding visual clutter. Prioritize vehicle and contact-related icons.
- Implement a large, impactful hero banner on the homepage to capture immediate attention. Utilize a card-based layout for inventory display to ensure intuitive browsing and a 'mobile-first' experience. Incorporate a real image gallery for each vehicle for transparency, and maintain a simple, clear navigation structure.
- A sticky bottom mobile bar will house essential 'Call' and 'WhatsApp' buttons for instant access to communication on mobile devices.
- Subtle and purposeful transitions for page changes, filtering actions, and interactive elements. Avoid excessive animations to maintain a fast and professional user experience, focusing on fluidity and responsiveness.
